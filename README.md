# 🌍 WanderWise - Premium Travel Planning Dashboard

live demo here 
https://yazhl31.github.io/Wander-Wise/

A modern, feature-rich travel planning dashboard built with HTML5, CSS3, and JavaScript. WanderWise helps users discover destinations, compare hotels and flights, and plan their perfect itinerary with an intuitive, premium interface.

## ✨ Features

### 🎬 **Cinematic Destination Slider**
- Full-screen image slider with 6 premium destinations
- Smooth autoplay and custom navigation arrows
- Each slide features a beautiful Unsplash image and "explore" button

### 🔍 **Smart Search & Filtering**
- Real-time filtering of hotels and flights by destination
- Intelligent keyword matching across locations
- "No results" messaging with helpful suggestions
- Instant reset when search is cleared

### 🌓 **Dark/Light Theme**
- Toggle between elegant light and dark modes
- Persistent theme preference using localStorage
- Smooth transitions between themes
- Carefully curated color palettes for each mode

### 🌐 **Multi-language Support**
- English/Spanish toggle with one click
- Language preference saved in localStorage
- All UI elements translated including navigation, buttons, and labels
- Easy to extend with additional languages

### 🗺️ **Interactive Maps**
- Dynamic Leaflet maps for each destination
- Automatic map loading when city modal opens
- Custom markers with city names
- Proper map cleanup between views

### 🏨 **Hotel Listings**
- Curated hotel cards with ratings and reviews
- High-quality Unsplash images for each property
- Price information and review counts
- Data attributes for intelligent filtering

### ✈️ **Flight Options**
- Premium flight listings with carriers and prices
- Duration and stop information
- Price tags and quick details
- Integrated with search functionality

### 💼 **Travel Packages**
- Three tiered pricing options (Essential, Elevated, Prestige)
- Feature lists for each package
- Clean, modern card design
- Hover effects and transitions

### 📅 **Trip Timeline**
- Editable day-by-day itinerary
- Customize plan modal with input fields
- Save functionality with demo feedback
- Clean, visual timeline design

## 🛠️ **Technologies Used**

| Technology | Purpose |
